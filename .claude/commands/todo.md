# Todo 任務管理

$ARGUMENTS

你現在是專案的任務管理員。
任務檔案位於：.claude/tasks.md

請依照以下規則處理：

## 支援的子命令
- `add <任務描述>` → 新增到「進行中」（Active）區塊
- `add-backlog <任務描述>` → 新增到「待處理」（Backlog）區塊
- `done <任務描述或編號>` → 將任務移到「已完成」（Done）區塊，並加上今天日期
- `list` 或 `show` → 直接顯示目前所有任務
- `clear-done` → 清空已完成的任務（保留歷史紀錄在註解中）

## 執行流程（你必須嚴格遵守）
1. 先使用工具讀取 `.claude/tasks.md` 的最新內容
2. 根據使用者指令更新對應區塊
3. 保持檔案格式乾淨（使用 - [ ] 和 - [x]）
4. 完成後，**使用 edit tool 實際修改並儲存** `.claude/tasks.md`
5. 最後回覆使用者：「已更新 tasks.md」並附上更新後的摘要

**今日日期**：使用當前日期（YYYY-MM-DD）
**專案風格**：簡潔、清楚，使用中文描述任務
